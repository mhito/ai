# AI Execute Mode Guide

## Overview

Execute mode allows you to automatically execute commands generated by AI after a confirmation prompt. This feature is useful when you want to run commands directly without manually copying and pasting them.

## Usage

### Method 1: Using `!` at the end (Recommended)

```bash
ai "your question!"
```

This is the quickest way to enter execute mode. Just add `!` at the end of your question.

### Method 2: Using flags

```bash
ai -x "your question"
# or
ai --execute "your question"
```

### Method 3: Escape `!` to prevent execution

```bash
ai "your question\!"
```

If you want to include `!` in your question without triggering execute mode, escape it with `\!`.

### How It Works

1. **Generate Command**: AI generates a command based on your question
2. **Display Command**: The command is displayed in the terminal
3. **Confirmation Prompt**: You see: `run <command>? (Y/n):`
4. **Execute or Cancel**: 
   - Press **Enter** or type **Y** to execute the command
   - Type **n** to cancel and exit without executing

The command appears between angle brackets `< >` for better visibility.

### Examples

#### Example 1: Using `!` (Recommended)
```bash
$ ai "list all files!"
ls -la
run <ls -la>? (Y/n): 
```
Press Enter to execute, or type `n` to cancel.

#### Example 2: Create a directory with `!`
```bash
$ ai "create a directory called my_project!"
mkdir my_project
run <mkdir my_project>? (Y/n): 
```

#### Example 3: Escape `!` to prevent execution
```bash
$ ai "find files with exclamation\!"
find . -name "*\!"
# (no execution prompt, just shows the command)
```

#### Example 4: Using `-x` flag
```bash
$ ai -x "find all python files"
find . -name "*.py"
run <find . -name "*.py">? (Y/n): 
```

## Safety Considerations

- **Always review the command** before pressing Enter
- The prompt defaults to **Y** (execute), so be careful
- For destructive operations (rm, dd, etc.), always type `n` if unsure
- The command is executed with your current user permissions

## Comparison with Regular Mode

### Regular Mode (without -x)
```bash
$ ai "show disk usage"
df -h
# You must manually copy and paste the command
```

### Execute Mode (with -x)
```bash
$ ai -x "show disk usage"
df -h
run df -h? (Y/n): 
# Press Enter to execute directly
```

## Tips

1. **Test first**: Use regular mode first to see what command will be generated
2. **Read carefully**: Always read the command before confirming execution
3. **Use for safe operations**: Best for non-destructive commands
4. **Combine with pipes**: You can still pipe the output:
   ```bash
   ai -x "list files" | grep "pattern"
   ```

## Troubleshooting

### Command doesn't execute
- Make sure you pressed Enter or typed `Y` (uppercase or lowercase)
- Check that the command is valid for your system

### Wrong command generated
- Type `n` to cancel
- Try rephrasing your question
- Use regular mode to preview before using `-x`

## Integration with Shell Aliases

You can create aliases for common execute mode commands:

```bash
# Add to ~/.bashrc
alias ai-x='ai -x'
```

Then use:
```bash
ai-x "your question"
```
